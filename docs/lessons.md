# Lessons

このドキュメントには、AI エージェントと人間が協働する中で得た教訓を記録する。

同じ指摘、同じ注意、同じ失敗、同じ確認事項を繰り返さないために、セッションをまたいで参照すべき内容を残す。

個人のメモリやチャット履歴には依存せず、チームでレビュー可能な共有コンテキストとして管理する。

不要になった教訓は、Git の履歴に任せて削除する。

## 教訓

- ドキュメント構成を変更する場合は、テンプレートとして残す意図があるかを確認する。
- 分類やフォルダを削除する場合は、単純化の利点とテンプレート性の損失を比較する。
- 方針が固まっていない構成変更では、実装を急がず、先に認識をすり合わせる。
- 教訓には、再発防止に役立つ作業ルールだけを記録する。今回限りのディレクトリ名、ドキュメント名、表現の選択理由は、該当ドキュメント側に反映し、`docs/lessons.md` には記録しない。
