# 設定・辞書仕様

## 基本方針

- 設定形式には TOML を使用する。
- 既定の設定ファイルは `~/.config/bosoboso/config.toml` とする。
- `--config`によって別の設定ファイルを指定できる。
- 相対パスは、パスを記述した設定ファイルの配置ディレクトリを基準に解決する。
- 設定ファイルが存在しない場合は自動生成せず、エラー画面へ場所と設定例を表示する。
- 設定と辞書には`version = 1`を必須とする。
- 未対応バージョン、未知の項目、不正な型、範囲外の値をエラーにする。
- 辞書以外の設定変更は、アプリ再起動後に反映する。

## 設定項目

次の項目を設定対象とする。正式なキー名と既定値は、技術検証と設定型の実装時に確定する。

| 分類 | 項目 | 方針 |
| --- | --- | --- |
| 形式 | 設定バージョン | `1`を必須とする |
| モデル | Whisper モデルパス | 必須。絶対パスと相対パスを許可する |
| 辞書 | 辞書ファイルパス | 必須。設定とは別ファイルにする |
| 認識 | 言語 | 初期値は日本語。変更可能にする |
| 認識 | Metal GPU | 初期状態で有効。無効化可能にする |
| 認識 | 前文脈の引き継ぎ | 初期状態で有効。無効化可能にする |
| VAD | 発話区間を閉じる無音時間 | 初期候補は 1.2 秒 |
| VAD | 最大発話区間長 | 初期候補は 30 秒 |
| VAD | 区間間の重複時間 | 技術検証後に初期値を確定する |
| 表示 | 新しい文章ブロックへ切り替える無音時間 | 初期候補は 10 秒 |
| 表示 | 保持する文章ブロック数 | 初期値は 100 件 |
| キュー | 処理待ち音声の上限 | 初期値は 60 秒 |

Apple Silicon では Metal を標準で使う。Core ML 用の追加モデルと Apple Neural Engine 対応は MVP に含めない。

## 辞書

辞書は、正規表記、読み方、明示的な誤認識候補を持つ。

```toml
version = 1

[[entries]]
term = "mise"
readings = ["みーず"]
misrecognitions = ["水", "ミーズ"]
```

### 適用規則

- `term`と`readings`は、Whisper の初期プロンプトによる認識誘導へ使用する。
- Whisper は発音辞書を直接扱わないため、読み方と正規表記の対応を保証するものではない。
- `misrecognitions`は、利用者が明示した候補だけを確定結果へ適用する。
- MVP では Unicode を正規化し、大文字と小文字を区別して完全一致する箇所だけを置換する。
- 複数の候補が重なる場合は、最も長い候補を優先する。
- 曖昧一致と編集距離による置換は行わない。
- 辞書全体が Whisper の初期プロンプト上限へ収まらない場合は、項目を黙って省略せず、読み込みエラーにする。

一致規則は、実際の認識精度を評価した後に変更する可能性がある。
