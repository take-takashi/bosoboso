# 用語集

| 用語 | 意味 |
| --- | --- |
| `bosoboso` | 本アプリの正式表記。先頭を含め、常に小文字で記載する |
| VAD | Voice Activity Detection。音声から発話区間と無音区間を検出する処理 |
| 発話区間 | VADによって切り出し、Whisperへ渡す内部処理単位 |
| 文章ブロック | 一つ以上の発話区間を連結した、利用者が編集・コピーする表示単位 |
| 暫定結果 | 発話中に更新される編集不可の文字起こし結果 |
| 確定結果 | 発話区間の処理と辞書補正が完了した文字起こし結果 |
| 認識誘導 | 辞書の正規表記と読み方をWhisperの初期プロンプトへ渡し、候補へ出やすくすること |
| 明示的置換 | 利用者が辞書へ登録した誤認識候補を、正規表記へ決定的に置換すること |
| 完全ローカル | 音声取得、文字起こし、補正を端末内で実行し、実行中にネットワーク通信しないこと |
| GUIコンポーネントテスト | Fyneのテストドライバーで、実ウィンドウや実マイクを使わずにGUIの振る舞いを検証するテスト |
| Integration Test | 音声ファイル、VAD、実Whisperモデル、辞書補正を接続して検証するローカルテスト |
| E2Eテスト | パッケージ済みアプリへ疑似音声を入力し、画面表示までを検証するローカルテスト |
