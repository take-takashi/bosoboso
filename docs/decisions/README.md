# Decision Records

このディレクトリには、重要な判断とその理由を記録する。

設計判断、技術選定、運用方針の変更など、後から経緯を確認したい内容を対象とする。

厳密な ADR 形式に限定せず、テンプレートとして扱いやすい粒度で記録する。

## Decision Records

- `0001-local-desktop-architecture.md`: 完全ローカルの単一プロセス構成
- `0002-external-whisper-model-and-dictionary.md`: 外部Whisperモデルと辞書補正
- `0003-fyne-and-test-boundaries.md`: Fyne採用と自動・実機テストの境界
