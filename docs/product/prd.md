# bosoboso PRD

## 概要

`bosoboso` は、起動中にマイクから取得した音声を完全にローカルで継続的に文字起こしし、利用者が必要な文章を小窓からクリップボードへコピーできるデスクトップアプリである。

最初の対象は、Codex や Claude Code などへ日本語のプロンプトを入力する開発者とする。日本語の発話に含まれる `mise` などの英字の製品名、サービス名、専門用語が誤認識される問題を、認識誘導と明示的な辞書補正によって軽減する。

## 解決する課題

- 長い日本語プロンプトをキーボードで入力する負担が大きい。
- 一般的な音声入力では、英字の製品名、サービス名、専門用語の表記が崩れやすい。
- 誤認識を毎回手で直す作業が、音声入力による効率化を損なっている。
- 音声やプロンプトをクラウドへ送信したくない。

## 対象利用者

- Apple Silicon 搭載 Mac を利用している開発者
- Codex や Claude Code などへ、考えながら日本語でプロンプトを入力する利用者
- 音声データと文字起こし結果を端末外へ送信または保存したくない利用者

## 提供価値

- アプリを起動している間の独り言や周囲の会話を、開始操作なしで継続的に文字化できる。
- 考えるために数秒間無言になっても、複数の発話を一つの文章ブロックとして扱える。
- 辞書によって、利用者固有の英字表記や専門用語を認識候補へ誘導できる。
- 必要な文章ブロックまたは選択範囲だけを、任意のアプリへ貼り付けるためにコピーできる。
- 音声、文字、利用状況を保存または送信しない。

## MVP のスコープ

### 含むもの

- macOS 14 以降かつ Apple Silicon で動作するデスクトップアプリ
- 選択したマイクからの常時音声取り込み
- 一時停止と再開
- Silero VAD による発話区間の検出
- 利用者が指定した `whisper.cpp` 対応モデルによるローカル文字起こし
- 発話中の暫定結果と、発話区間処理後の確定結果の表示
- 辞書による認識誘導と、明示的な誤認識候補の置換
- 複数の認識区間をまとめた文章ブロック
- 文章ブロックの編集、本文コピー、任意範囲のコピー
- 最新の文章ブロックを上にした履歴表示
- 入力デバイスの選択と切断時の自動フォールバック
- 厳密な設定検証と明確なエラー表示

### 含まないもの

- クラウド API または任意のネットワーク通信
- 音声、文字起こし結果、ウィンドウ位置、運用ログの永続保存
- Whisper とは別の LLM による文章校正
- フィラーの自動削除
- 入力先アプリの検出または入力欄への自動貼り付け
- グローバルショートカット
- システム音声の直接取得
- ログイン時の自動起動、メニューバー常駐、バックグラウンド常駐
- Intel Mac、Windows、Linux の動作保証
- コード署名、公証、第三者向け配布

## 成功条件

- 起動中の発話を継続して文字起こしできる。
- 5 秒程度の無音を挟む発話を、同じ文章ブロックとして扱える。
- `mise`、`Codex`、`Claude Code`などの登録語を、認識誘導または明示的な置換によって期待する表記にできる。
- 確定した文章を小窓で修正し、本文だけを大きな手直しなしでコピーできる。
- 音声と文字を保存せず、実行中にネットワーク通信を行わない。
- Apple Silicon と macOS 14 以降で、日常利用できる遅延に収まる。

## 実機評価後に再検討する事項

- 暫定結果の更新間隔と、確定結果を表示するまでの目標時間
- VAD の無音判定時間、文章ブロックを切り替える無音時間
- Whisper の前文脈を引き継ぐ効果と誤認識の連鎖
- 辞書置換に必要な一致規則
- 別のローカル LLM による校正の必要性
- グローバルショートカットと入力欄への貼り付け支援
